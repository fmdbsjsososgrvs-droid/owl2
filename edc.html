// 추천 결론

  • 🥇 1순위 — COLDCARD Q 최고 보안, 듀얼 보안칩, 완전 에어갭. 비트코인 보관의 표준
  • 🥈 2순위 — SEEDSIGNER 100% 오픈소스 DIY, 5만원의 가성비. 상태비저장 설계로 최고의 프라이버시
COLDCARD Q RANK 1 AIR-GAPPED · DUAL CHIP
  • 최고 보안 수준 — 비트코인 전용 설계, 듀얼 보안칩
  • 완전 에어갭 모드 지원 (SD카드 / NFC / QR) — USB 없이도 PSBT 서명 가능
  • 피싱 방어 기능 — Anti-Phishing Words, Trick PIN, BIP-39 Passphrase 내장
  • QWERTY 키보드 탑재 (Q 모델 한정) — 길고 복잡한 패스프레이즈도 빠르게 입력
SEEDSIGNER RANK 2 DIY · STATELESS
  • 완전 오픈소스 DIY 지갑 — Raspberry Pi Zero 기반, 직접 조립 가능
  • 약 5만원의 저렴한 가격 — 부품 가격만으로 구성, 진입 장벽 낮음
  • QR코드 에어갭 방식 — USB·블루투스·네트워크 일체 없음
  • 상태비저장(stateless) 설계 — 시드를 기기에 저장하지 않음, 매번 입력 → 분실해도 시드 유출 위험 0
KEYSTONE QR-BASED
  • 장점 — 한글화 지원으로 한국 사용자 친화적, 큰 컬러 화면, QR 에어갭
  • 단점 — 중국산 (제조 신뢰성·공급망 우려), 일부 오픈소스가 아닌 모듈 존재
JADE BLOCKSTREAM
  • Blockstream이 만든 오픈소스 콜드월렛, QR 에어갭 모드 지원
  • 합리적인 가격대(약 12만원), 멀티시그·리퀴드 통합 지원
  • 한글화 미흡, COLDCARD/SEEDSIGNER 대비 보안 명성은 약간 부족
BITKEY BLOCK · 2-OF-3
  • 장점 — 이쁘다. 디자인이 잘 빠진 하드웨어, 모바일 앱 UX가 뛰어남
  • 단점 — 완전한 셀프 수탁이 아니다. Block(Square) 서버가 키 1개를 보관하는 2-of-3 멀티시그 구조로 공급자 의존 발생
02 보안 키 / 카드형 디바이스

온라인 계정과 모바일 비트코인을 지키는 물리적 보안 도구들.

YUBIKEY SECURITY KEY · 2FA
  • 물리적 보안 키. 비밀번호를 알아도 키 없이는 계정에 접근 불가
  • 펜타곤, 미 국무부 등 고위험 기관이 채택한 최상급 보안 수준
  • X, Google, Apple, Discord, Binance, ProtonMail 등 주요 서비스 지원
  • 개당 약 100,000원. Apple 등록 시 2개 필수 — 최소 200,000원 투자
TAPSIGNER NFC SIGNER · COINKITE
  • Coinkite가 만든 NFC 카드형 시그너. 스마트폰에 갖다 대면 PSBT에 서명
  • 두께 1mm의 일반 신용카드 크기 — 지갑·여권 사이에 꽂아 휴대 가능
  • 개봉 후 한 번만 시드 생성, 칩 내부에서 절대 외부로 노출되지 않음
  • Nunchuk, Sparrow 등 주요 와치온리 앱과 호환
SATSCARD DISPOSABLE · BEARER
  • Coinkite의 일회용 비트코인 카드. 카드 자체가 곧 지갑(bearer instrument)
  • 슬롯 10개 내장 — 한 슬롯을 "unseal"하면 그 슬롯의 비트코인을 꺼낼 수 있음
  • 실물 선물·팁 카드로 활용 — NFC로 잔액 즉시 확인 가능
  • 긴 보관용이 아닌 단기 휴대·증여용으로 적합
03 라이트닝 & 지갑

일상 결제와 온체인-라이트닝 자유 이동을 위한 도구들.

CORN WALLET LIGHTNING · KOREA
  • 한국 비트코이너 팀이 만든 라이트닝 지갑. 레트로 아이콘과 팝콘 애니메이션의 귀여운 디자인
  • 직관적인 메뉴 구성 — 필요한 기능만 담은 깔끔한 UX
  • 단 0.49%의 경쟁력 있는 온체인 수수료
  • 저렴하고 편리한 라이트닝 결제 지갑
BIT FLOWER TELEGRAM · ON→LN SWAP
  • 텔레그램 기반의 라이트닝 스왑 도구. 온체인 BTC를 라이트닝으로 이동시켜주는 간편한 서비스
  • 텔레그램 봇으로 주문서 생성 — 와치온리 앱 & 콜드월렛으로 서명 후 실행
  • QR 코드 방식으로 안전하고 간단한 자산 이동 과정
  • 비트코인 라이트닝 네트워크 진입을 위한 실용적인 도구
AQUA WALLET LIQUID L2 · PRIVACY
  • 리퀴드 비트코인 특화 L2 지갑. 빠르고 저렴한 비트코인 거래 지원
  • 온체인 BTC를 리퀴드 BTC로 스왑 — 거래소 추적을 일시적으로 회피 가능
  • 라이트닝 & 리퀴드 자산 보관, 자동 변환 기능 내장
  • 깔끔한 디자인과 실용성을 갖춘 비트코인 프라이버시 도구
COCONUT WALLET WATCH-ONLY · MULTISIG
  • 와치온리 전용 모바일 지갑. 콜드월렛에 보관된 키를 안전하게 관찰·관리
  • COLDCARD·SEEDSIGNER·KEYSTONE 등 주요 콜드월렛과 QR / SD 에어갭 호환
  • 멀티시그 지원 — 가족·동료와 함께하는 협업 수탁(collaborative custody)에 최적
  • 송금 시 PSBT 생성 → 콜드월렛으로 서명 → 다시 앱에서 브로드캐스트하는 완전 분리 워크플로우
NUNCHUK WATCH-ONLY · COLLAB CUSTODY
  • 멀티시그·셀프 수탁 특화 와치온리 앱. 모바일 + 데스크톱 모두 지원
  • 2-of-3, 3-of-5 등 다양한 멀티시그 구성을 GUI로 손쉽게 설정
  • 주요 하드웨어 지갑(COLDCARD, JADE, KEYSTONE, Trezor, Ledger) 모두 호환
  • 가족·신뢰 그룹과 키를 분산해 보관하는 협업 수탁의 표준 도구
